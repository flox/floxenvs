# CodeBurn Demo

Interactive demo of the [CodeBurn](../codeburn/)
environment with a styled banner.

## Quick start

```bash
flox activate -r flox/codeburn-demo
codeburn
```

## What it provides

Everything from [codeburn](../codeburn/) plus:

- `gum` — styled terminal UI
- Welcome banner on activate
