flox / envs

Ready-to-use
Flox envs and packages.

One include line. AI agents, runtimes, plugins, skills, and the glue to make them work together.

flox/envs is the curated catalog of Flox environments and packages — reproducible dev setups for AI coding agents (Claude Code, Codex, Crush), local LLM runtimes (Ollama), RAG stacks (LangChain, PostgreSQL + pgvector, Verba), and the languages, databases, and services around them. Every entry is Nix-pinned, included with a single manifest line, and reproducible across every developer's machine and CI runner. Published by Flox, MIT-licensed catalog source.

Browse AI items → Search everything 
# my-project/.flox/env/manifest.toml
[include]
environments = [
  { remote = "flox/claude" },
  { remote = "flox/postgresql" },
  { remote = "flox/langchain" },
]

[install]
ollama.pkg-path     = "flox/ollama"
ollama.publisher    = "flox"
