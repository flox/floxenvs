# hermes-demo

Interactive demo of the [hermes](../hermes/) environment
with a styled welcome banner.

## Quick start

```bash
flox activate -r flox/hermes-demo
hermes setup
```

## What it provides

Everything from [hermes](../hermes/) plus:

- `gum` — styled terminal UI
- Welcome banner on activate
