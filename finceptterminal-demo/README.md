# finceptterminal-demo

Demo walkthrough for the
[finceptterminal](../finceptterminal/) env. Adds a styled gum
banner explaining the launch command and where data lives.

## Quick start

```bash
flox activate -r flox/finceptterminal-demo
finceptterminal
```

The banner that appears on activation summarises everything you
need to know to launch the app.

## What is included

| Package | Description |
| ------- | ----------- |
| `flox/finceptterminal` (via `[include]`) | The minimal env. |
| `gum` | Used for the styled banner. |
