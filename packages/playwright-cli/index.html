Playwright CLI

Microsoft Playwright CLI for recording, inspecting, and driving browsers from the shell.

  • Record browser sessions and generate Playwright test code
  • Inspect CSS selectors and debug web pages interactively
  • Works with browsers provided by env:playwright at the env layer

Install

[install]
playwright-cli.pkg-path = "flox/playwright-cli"
playwright-cli.publisher = "flox"
