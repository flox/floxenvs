# Vibe Kanban Demo

<!-- codespaces-badge -->
[![Open in Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/flox/floxenvs?devcontainer_path=.devcontainer%2Fvibe-kanban-demo%2Fdevcontainer.json)

Interactive demo of the [Vibe Kanban](../vibe-kanban/)
environment with a styled banner.

## Quick start

```bash
flox activate -r flox/vibe-kanban-demo
vibe-kanban
```

Then open the printed URL in your browser.

## What it provides

Everything from [vibe-kanban](../vibe-kanban/) plus:

- `gum` — styled terminal UI
- Welcome banner on activate
